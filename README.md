# 🍰 Página de Receita

Projeto de uma página web estática de receita, desenvolvida com foco em estruturação semântica em HTML e estilização com CSS.

---

## 📌 Sobre o Projeto

A Página de Receita é um projeto front-end criado para praticar a construção de layouts organizados utilizando apenas HTML e CSS.

O objetivo foi aplicar boas práticas de marcação semântica, organização de conteúdo e estilização visual, criando uma interface clara e agradável para o usuário.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

---

## 🎯 Funcionalidades

- Estrutura semântica organizada
- Seção de ingredientes e modo de preparo
- Layout centralizado e limpo
- Estilização personalizada

---

## 🧠 Conceitos Praticados

- Tags semânticas (`header`, `main`, `section`, `article`, `footer`)
- Organização de conteúdo
- Seletores CSS
- Box Model
- Flexbox

---

## 💡 Objetivo

Praticar fundamentos sólidos de HTML e CSS, reforçando conceitos essenciais para desenvolvimento front-end.

---

## 📂 Como Executar o Projeto

1. Clone este repositório: https://github.com/Matheusoliveira06/pagina-de-receita.git   
2. Abra o arquivo `index.html` no navegador.
